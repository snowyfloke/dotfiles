# Giphy Search

A launcher provider plugin that allows you to search for GIFs on Giphy directly from the launcher, with animated previews in a grid layout.

## Features

- **GIF Search**: Search Giphy's library directly from the launcher
- **Animated Previews**: See GIF thumbnails in a grid layout
- **Copy to Clipboard**: Select a GIF to copy its URL to your clipboard
- **Content Rating Filter**: Filter results by rating (G, PG, PG-13, R)
- **Command Mode**: Use `>gif` for dedicated GIF search

## Usage

1. Open the launcher
2. Type `>gif` followed by your search query
3. Browse the grid of GIF previews
4. Press Enter to copy the selected GIF's URL to your clipboard


## Configuration

- **Giphy API Key**: Required. Get a free key from [developers.giphy.com](https://developers.giphy.com)
- **Content Rating**: Filter results by rating (G, PG, PG-13, R)
